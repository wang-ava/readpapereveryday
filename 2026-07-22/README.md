# 2026-07-22 AI 论文分享

## 当日推荐（按关注度排序）

1. **Copy Less, Ground More: Overcoming Repetitive Copying in Long-Context Reasoning via Evidence-Aware Reinforcement Learning**（重点关注）  
   - Spotlight：首次将 long-context 推理中的重复抄写失败显式建模并提出 grounding-aware 奖励，直接关系 LLM 的可靠性与可控性。
   - 链接： https://arxiv.org/abs/2607.19345v1

2. **CodeRescue: Budget-Calibrated Recovery Routing for Coding Agents**  
   - Spotlight：把 coding agent 失败后处理建成预算可控路由问题，给出提升成功率同时降本的实际操作思路。
   - 链接： https://arxiv.org/abs/2607.19338v1

3. **OmniReasoner: Thinking with Long Audio-Video via Native Tool Use**  
   - Spotlight：长音视频理解场景下的“全局预览+局部放大”工具调用策略，有效聚焦关键信息、节省计算。
   - 链接： https://arxiv.org/abs/2607.19339v1

4. **Masked Visual Actions for Unified World Modeling**  
   - Spotlight：以可视化动作掩码统一世界模型前向与逆向建模，兼顾控制执行与轨迹解释性，面向具身 AI 有参考价值。
   - 链接： https://arxiv.org/abs/2607.19343v1

5. **BioSecBench-Surveillance: A Verifiable Benchmark for AI Agents in Pathogen Genomic Surveillance**  
   - Spotlight：AI4S 高风险场景的重要基准化尝试，把“跑得通”与“跑对”区分开，强调科学决策链条验证。
   - 链接： https://arxiv.org/abs/2607.19262v1
