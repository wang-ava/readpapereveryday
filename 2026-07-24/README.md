## 当日推荐（按关注度排序）

1. **AREX: Towards a Recursively Self-Improving Agent for Deep Research（重点）**
   - Spotlight：将深度研究从“持续搜索”升级为“验证后自我改进”，适合关注 Agent 可靠性和研究自动化。
   - 链接： https://arxiv.org/abs/2607.21461v1

2. **S1-Omni: A Unified Multimodal Reasoning Model for Scientific Understanding, Prediction, and Generation**
   - Spotlight：AI4S 场景下的统一建模尝试，将异构科学数据与科学任务统一到同一推理框架。
   - 链接： https://arxiv.org/abs/2607.15686v1

3. **ATSplat: Compact Feed-forward 3D Gaussian Splatting with Adaptive Token Expansion**
   - Spotlight：通过 adaptive token 机制把 feed-forward 3DGS 从像素对齐改为场景复杂度驱动，显著压缩高斯规模并加快渲染。
   - 链接： https://arxiv.org/abs/2607.20417v1

4. **VoLN: Vision-Only Long-Horizon Navigation---Paradigm, Benchmark, and Method**
   - Spotlight：把长时导航从路线先验转为视觉本地线索驱动，明确暴露具身智能在复杂环境中的真实瓶颈。
   - 链接： https://arxiv.org/abs/2607.21400v1

5. **Notes to Self: Can LLMs Benefit from Experiential Abstractions?**
   - Spotlight：探索 LLM 的经验抽象可复用性，支持“经验性记忆”向可检索策略转换。
   - 链接： https://arxiv.org/abs/2607.20372v1
