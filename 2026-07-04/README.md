# 2026-07-04 AI 论文推荐

本目录汇总当天检索的 5 篇近期 AI 论文（优先筛选 2026-07-04 当地时间前 7–14 天内，覆盖 LLM/Agent/CV/具身 AI/AI4S）。

1. 实用价值
2. 方法新意
3. 可复现性与工程落地线索
4. 主题覆盖均衡

## 推荐列表（含 spotlight 与链接）

1. [LACUNA: A Testbed for Evaluating Localization Precision for LLM Unlearning](https://arxiv.org/abs/2607.02513v1)
   - Spotlight：将 unlearning 评测从“输出好不好看”推进到参数级证据，解决安全审计中的误判风险。
   - 文件：`LACUNA-LLM参数定位卸载.md`

2. [Program-as-Weights: A Programming Paradigm for Fuzzy Functions](https://arxiv.org/abs/2607.02512v1)
   - Spotlight：把重复 LLM 工程任务从在线推理转为可复用神经程序，降低成本并提升部署确定性。
   - 文件：`PAW-程序即权重编译.md`

3. [WorldDirector: Building Controllable World Simulators with Persistent Dynamic Memory](https://arxiv.org/abs/2607.02517v1)
   - Spotlight：通过持久动态记忆把世界模拟从“尽力生成”改为“可控、可复现地演化”。
   - 文件：`WorldDirector-可控世界模拟器.md`

4. [Embodied.cpp: A Portable Inference Runtime of Embodied AI Models on Heterogeneous Robots](https://arxiv.org/abs/2607.02501v1)
   - Spotlight：把 VLA/WAM 部署难题落到 runtime 设计，强调多速率闭环与硬件异构一致性。
   - 文件：`Embodied.cpp-异构机器人模型运行时.md`

5. [AgentsCAD: Automated Design for Manufacturing of FDM Parts via Multi-Agent LLM Reasoning and Geometric Feature Recognition](https://arxiv.org/abs/2607.02448v1)
   - Spotlight：面向 AI4S 的多智能体制造工作流案例，目标是让几何设计具备“自动修订”能力。
   - 文件：`AgentsCAD-多智能体制造设计.md`
