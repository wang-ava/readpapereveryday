# 2026-07-12 AI 论文分享

## 当日推荐（按关注度排序）

1. **Generative Skill Composition for LLM Agents**（重点关注）  
   - Spotlight：把 skill 使用从“检索”升级为“结构化生成”，一次预测技能子集、数量与顺序，显著提升编程 agent 的长任务成功率。  
   - 链接：https://arxiv.org/abs/2606.32025

2. **AgentLens: Production-Assessed Trajectory Reviews for Coding Agent Evaluation**  
   - Spotlight：提出轨迹级评测框架，用 formal checks + 轨迹评审解释 coding agent 的全过程行为，而非只看是否通过。  
   - 链接：https://arxiv.org/abs/2607.06624

3. **When Agents Do Not Stop: Uncovering Infinite Agentic Loops in LLM Agents**  
   - Spotlight：用静态分析定义并检测 IAL（Infinite Agentic Loops），覆盖真实项目规模仓库，直面 agent 无限执行风险。  
   - 链接：https://arxiv.org/abs/2607.01641

4. **WildCity: A Real-World City-Scale Testbed for Rendering, Simulation, and Spatial Intelligence**  
   - Spotlight：构建城市级真实多模态数据与 closed-loop simulator，提供 space/embodied reasoning 的长时空评测入口。  
   - 链接：https://arxiv.org/abs/2607.06838

5. **3D Reconstruction of Deciduous Trees using low-cost UAV- and Crane-based Photogrammetry for Monitoring Shoot Elongation across entire Canopies**  
   - Spotlight：面向 AI4S 的低成本城市生长监测路线，强调树冠级别 3D 重建可重复性与细枝重建评估。  
   - 链接：https://arxiv.org/abs/2607.07905
