## 📖 阅读建议

### 如果你是完全的新手
推荐阅读顺序：
1. **Transformer**（理解所有现代AI的基础）
2. **CLIP**（理解多模态AI）
3. **DDPM**（理解生成式AI）
4. **InstructGPT**（理解为什么ChatGPT"听话"）
5. **Chain-of-Thought**（理解提示工程）

### 如果你对AI4Health感兴趣
推荐阅读顺序：
1. **ClinicalBERT**（经典，理解医疗NLP基础）
2. **BiomedCLIP**（理解医疗多模态）
3. **RETFound / GigaPath**（理解医疗影像基础模型）
4. **Med-PaLM 2 / Med-Gemini**（理解医疗LLM的前沿）
5. **AlphaFold 3**（理解AI在药物研发中的应用）

### 如果你想快速了解大模型
推荐阅读顺序：
1. **Transformer** → **GPT-4** → **LLaMA 3** → **LoRA** → **Mixtral**
