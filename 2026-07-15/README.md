# 2026-07-15 AI 论文分享

## 当日推荐（按关注度排序）

1. **Requential Coding: Pushing the Limits of Model Compression with Self-Generated Training Data**（重点关注）  
   - Spotlight：将压缩问题从参数层面转向“学习分歧信息编码”，在理论与实证上都展示了更强尺度外推能力。  
   - 链接：https://arxiv.org/abs/2607.11883

2. **Read It Back: Pretrained MLLMs Are Zero-Shot Reward Models for Text-to-Image Generation**  
   - Spotlight：把预训练 MLLM 直接用于 zero-shot reward，降低文本到图像 RL 的偏好数据依赖，强调可复用性和闭环自改进。  
   - 链接：https://arxiv.org/abs/2607.11886

3. **MM-ToolSandBox: A Unified Framework for Evaluating Visual Tool-Calling Agents**  
   - Spotlight：首次系统化评测“视觉+多轮+状态变更”下的 tool-calling agent，真实暴露了当前系统可用性瓶颈。  
   - 链接：https://arxiv.org/abs/2607.11818

4. **Mixture of Frames Policy: Multi-Frame Action Denoising for Bimanual Mobile Manipulation**  
   - Spotlight：用多坐标系扩散去噪处理 bimanual 操作中的几何不一致问题，对真实具身任务很有迁移价值。  
   - 链接：https://arxiv.org/abs/2607.11884

5. **Metacognition in LLMs: Foundations, Progress, and Opportunities**  
   - Spotlight：给 LLM metacognition 构建了方法论地图，尤其对可靠性、可反思能力和评测标准化具有直接价值。  
   - 链接：https://arxiv.org/abs/2607.11881
