# 2026-07-11 AI 论文分享

## 当日推荐（按关注度排序）

1. **UniClawBench: A Universal Benchmark for Proactive Agents on Real-World Tasks**
   - Spotlight：提出能力驱动的 proactive agent benchmark，强调动态闭环评测和失败归因，可作为真实世界智能体评测基线。
   - 链接：https://arxiv.org/abs/2607.08768v1

2. **Super Weights in LLMs and the Failure of Selective Training**
   - Spotlight：反驳“只训最重要权重”直觉，提醒参数重要性与可训练性不可混淆。
   - 链接：https://arxiv.org/abs/2607.08733v1

3. **Remember When It Matters: Proactive Memory Agent for Long-Horizon Agents**
   - Spotlight：提出 plug-and-play 的主动记忆模块，在长程任务 benchmark 上显著提分。
   - 链接：https://arxiv.org/abs/2607.08716v1

4. **ARDY: Autoregressive Diffusion with Hybrid Representation for Interactive Human Motion Generation**
   - Spotlight：用混合表示和自回归扩散实现文本+约束驱动的人体动作实时交互生成，兼顾质量与控制。
   - 链接：https://arxiv.org/abs/2607.08741v1

5. **LTM: Large-scale Terrain Model for Wildfire-prone Landscapes**
   - Spotlight：面向 AI4S 的低成本火险地形重建路线，使用旧 DEM 先验提高野火区域建模效率。
   - 链接：https://arxiv.org/abs/2607.08711v1
