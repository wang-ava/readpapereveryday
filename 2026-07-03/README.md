# 2026-07-03 AI 论文推荐

本目录汇总当天筛选的 5 篇近期论文（优先覆盖过去一周内发布、近 7–14 天窗口）。顺序按“可读性、实用性、跨主题覆盖”综合排序。

1. [AutoMem: Automated Learning of Memory as a Cognitive Skill](https://arxiv.org/abs/2607.01224)
   - 亮点：把 memory management 作为可学习技能双循环优化，2x–4x 提升长时序表现。
   - 说明：适合关注 agent memory 与长期决策稳定性。
   - 文件：`AutoMem-内存即能力.md`

2. [Can Agents Generalize to the Open World? Unveiling the Fragility of Static Training in Tool Use](https://arxiv.org/abs/2607.01084)
   - 亮点：将 open-world 条件下工具代理泛化问题做了分层建模与扰动增强训练。
   - 说明：适合关注 agent 部署鲁棒性与评测框架。
   - 文件：`CanAgentsGeneralize-OpenWorld.md`

3. [FurnitureVLA: Learning Long-Horizon Bimanual Furniture Assembly with Vision-Language-Action Model](https://arxiv.org/abs/2607.01212)
   - 亮点：提出真实尺度双臂家具装配基准，进度引导 VLA 显著改善长任务成功率。
   - 说明：具身 AI 的长期任务建模与 sim2real 对照很有借鉴性。
   - 文件：`FurnitureVLA-具身双臂装配.md`

4. [PointSplat: Compact Gaussian Splatting via Human-Centric Prediction](https://arxiv.org/abs/2606.32036)
   - 亮点：人本位三维点集预测减少冗余，提高实时渲染效率。
   - 说明：适合关注 CV / 实时 3D 表达优化。
   - 文件：`PointSplat-人景高效高斯重建.md`

5. [Graph-Native Reinforcement Learning Enables Traceable Scientific Hypothesis Generation through Conceptual Recombination](https://arxiv.org/abs/2607.00924)
   - 亮点：图原生 RL 提升科学假设生成可追溯性，材料科学任务上有 40–65% 改善。
   - 说明：AI4S 方向可验证性与解释性研究的代表范式。
   - 文件：`GraphNative-GRPO-AI4S.md`
